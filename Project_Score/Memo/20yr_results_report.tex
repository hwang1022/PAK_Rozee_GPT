\section*{1.\quad Overview}

This report presents the results of the RozeeAI difference-in-differences analysis adapted for the full 20-year Rozee.pk dataset (2004--2024, 740,657 job postings). The analysis follows the same empirical strategy as the ADB presentation (Ramachandran \& Garcia-Mandic\'{o}), estimating the impact of public-use ChatGPT (launched November 2022) on labor demand and wages by occupation-level AI exposure.

\noindent \textbf{Key differences from the ADB analysis:}
\begin{itemize}
    \item \textbf{Sample:} 740K jobs (2004--2024) vs.\ 338K (2020--2025)
    \item \textbf{Pre-period:} 18 years vs.\ $\sim$3 years
    \item \textbf{SOC matching:} Claude Haiku title $\to$ SOC code (single pass) vs.\ GPT-4 three-level matching
    \item \textbf{Salary:} Combined \texttt{sal\_from\_num} + \texttt{sal\_from\_hide\_num}, p99 trimmed
    \item \textbf{ISIC sector:} Not available in 20-year data; skipped
\end{itemize}

\section*{5.\quad Notes}

\begin{itemize}
    \item \textbf{Estimation equation:} $Y_{it} = \beta_0 + \beta_1 \text{LLM\_Exposure}_i + \beta_2 \text{Post}_t + \beta_3 (\text{LLM\_Exposure}_i \times \text{Post}_t) + \varepsilon_{it}$, with occupation and month fixed effects absorbed via \texttt{reghdfe}.
    \item \textbf{AI exposure measures:} Eloundou et al.\ (2024) task-level labels (E0/E1/E2) aggregated using three scoring schemes: $\alpha$ (E1 only), $\beta$ (E1 + 0.5$\times$E2), $\gamma$ (E1 + E2). Both human annotator and GPT-4 ratings.
    \item \textbf{Post period:} November 2022 onward (public launch of ChatGPT).
    \item \textbf{Occupation unit:} Broad occupation (459 categories) from O*NET-SOC hierarchy, assigned via modal SOC code across matched tasks per job.
    \item \textbf{35K ADB-only jobs:} The ADB dataset contains 35,545 job IDs not present in the 20-year data. These are excluded by design.
    \item \textbf{SOC code validation rate:} $\sim$79\% of Claude Haiku SOC code returns matched a valid code in the O*NET labelset. Unmatched jobs are excluded.
\end{itemize}
